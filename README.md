# 一元二次方程计算器
一个基于python的一元二次方程求解的程序，输入ax^2+bx+c=0的参数a,b,c即可求出方程的解。
基于韦达定理的一元二次方程求解程序：
$$ax^2+bx+c=0$$
其中a , b , c 是输入参数
$$x = \frac{{-b \pm \sqrt{{b^2 - 4ac}}}}{{2a}}$$
